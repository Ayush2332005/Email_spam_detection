# Email Spam Classifier

## Description
This project classifies emails into Spam or Not Spam using NLP and Machine Learning.

## Technologies Used
- Python
- Scikit-learn
- NLTK
- TF-IDF

## How to Run

1. Create virtual environment
2. Install dependencies:
   pip install -r requirements.txt

3. Train model:
   python src/train.py

4. Run app:
   python app.py
